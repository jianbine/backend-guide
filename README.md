## 📖 简介

面向有 Node.js 基础的前端开发者，通过**对比学习法**系统掌握后端开发核心概念，快速迁移到 Java Spring Boot 技术栈。

涵盖 **13 个核心章节**，覆盖环境搭建、路由、中间件、数据库、认证、部署、消息队列、搜索引擎、缓存、GraphQL、微服务、测试等后端必备知识。

## 🛠️ 本地开发

### 环境要求

| 工具 | 版本 | 说明 |
|------|------|------|
| **Node.js** | ≥ 18.0 | 推荐 v20+ LTS |
| **npm** | ≥ 9.0 | 推荐 v10+ |

### 快速启动

```bash
# 克隆项目
git clone https://github.com/jianbine/backend-guide.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📚 技术栈

- **Vue 3** + Composition API
- **Vite** 构建
- **Vue Router** 路由
- **Pinia** 状态管理
- **markdown-it** 内容渲染
